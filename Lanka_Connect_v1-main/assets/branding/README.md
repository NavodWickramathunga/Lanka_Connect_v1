Place your provided LankaConnect images in this folder with these exact names:

1. `splash_background.png`
Use the full teal poster background image.

2. `splash_foreground.png`
Use the centered logo/text composition that should animate in the splash.

3. `app_icon.png`
Use the square app icon artwork (1024x1024 recommended, PNG, no transparency for iOS).

After adding `app_icon.png`, run:

`flutter pub get`
`flutter pub run flutter_launcher_icons`
